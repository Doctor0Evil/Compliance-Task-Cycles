```markdown
# IOM Webclient Starter

## Features
- Secret Comet browser commands for instant navigation!
- Submit new memories/fragments quickly
- Search (public + AI-powered) for lost music and video
- Confirm matches (OP or trusted solver)
- All data feeds the evolving IOM Memory-Bridge

## Comet Secret Commands
- `iom://find` – Search instantly from the address bar!
- `iom://submit` – Jump directly to memory submission!
- `iom://random` – Explore a random IOM mystery!

**Note:**
- Works in all modern browsers, but “magic” commands are exclusive to Perplexity Comet.

## Install / Run
- `npm install`
- `npm start`
```
